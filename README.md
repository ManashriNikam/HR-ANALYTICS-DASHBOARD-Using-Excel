<b>📊 HR Analytics Dashboard Using Excel</b></br>
📌<b> Project Overview</b></br>
The HR Analytics Dashboard is an interactive Microsoft Excel dashboard developed to analyze workforce data and provide actionable insights into employee demographics, attrition, job roles, education, departments, and overall HR performance. The dashboard enables HR professionals to make data-driven decisions through dynamic visualizations and KPI tracking.

🎯 <b>Objectives</b></br>
Monitor employee attrition and retention.</br>
Analyze workforce demographics.</br>
Identify departments and job roles with higher attrition.</br>
Understand the impact of education, age, and marital status on employee turnover.</br>
Support HR decision-making using interactive reports.</br>
